## lab1 — Bash script

## lab2 — Ansible playbook

Вторая домашняя работа: Ansible playbook для создания пользователя на удаленной машине, настройки sudo, SSH-доступа по ключу, отключения входа по паролю и создания директории в `/opt/`.

## lab3 — Ansible roles и Molecule

Третья домашняя работа: та же задача на Ansible, но реализованная через несколько ролей с переменными и тестированием через Molecule с Docker driver.

## lab4 — Dockerfile
